# Big Game Hub

Ein kleines, anfängerfreundliches Static-Web-Projekt für deine GitHub Pages rund um das „Big Game“.

## Features
- Dark/Light Mode Toggle (mit Speicherung in `localStorage`)
- Live-Countdown bis Kickoff (ISO-Datum in `script.js` konfigurierbar)
- Party-Checkliste mit Speicherfunktion
- Mini-Scoreboard (Demo) mit Plus/Minus und Reset

## Anleitung (GitHub Pages)
1. Neues Repository auf GitHub anlegen (z.B. `big-game-hub`).
2. Dateien in das Repo kopieren und committen:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `assets/`
3. In GitHub unter **Settings → Pages** einstellen:
   - **Source:** Deploy from branch
   - **Branch:** `main` / `/root`
4. Speichern und auf die bereitgestellte URL warten.

## Hinweis zu Markenrechten
Nutze neutrale Begriffe wie „Big Game“ und vermeide geschuetzte Logos oder Markennamen.
